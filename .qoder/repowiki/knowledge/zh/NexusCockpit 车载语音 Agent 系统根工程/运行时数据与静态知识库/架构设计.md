纯数据目录，按用途划分为若干子目录：`food/` 提供食物营养与餐厅信息的 JSON 数据集；`knowledge/` 存放 FAQ、故障排查、车辆手册、语音指令等 Markdown 文档；`preferences/default_user.json` 定义默认用户的音乐、饮食、位置、气候、导航等偏好结构；`uploads/` 与 `temp/` 通过环境变量 `UPLOAD_DIR` / `TEMP_DIR` 映射到外部路径，仅保留 `.gitkeep` 占位；`checkpoints/` 由 LangGraph 的 `SqliteSaver` 写入 `nexus_checkpoints.db`，以 `thread_id` 隔离不同会话状态，实现重启不丢失。根 `README.md` 统一说明各子目录及对应环境变量，除 `.gitkeep` 外均被 Git 忽略，避免把大体积数据纳入版本控制。
