### Next.js 前端框架
- **角色**：企业级车载语音 Agent 的前端界面，提供座舱控制台、聊天界面、数据中台看板
- **集成点**：`frontend_design/src/app/` 页面路由，Zustand 状态管理，Tailwind CSS 样式系统
- **技术栈**：Next.js 14 (App Router)、React 18、TypeScript、Three.js 3D 可视化
- **关键特性**：SSR/SSG 支持、文件路由、与 Go 网关 8080 端口集成、SSE 流式响应处理
