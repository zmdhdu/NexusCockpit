### Next.js 前端框架
- **角色**: 企业级车载语音 Agent 的前端界面，提供座舱控制和管理后台
- **集成点**: `frontend_design/src/app/` 基于 App Router 的文件路由系统
- **技术栈**: React 18 + TypeScript + Tailwind CSS + Zustand 状态管理
- **关键页面**: cockpit（座舱控制台）、chat（聊天界面）、dashboard（运营总览）、admin（管理后台）
- **构建优化**: SSR/SSG 支持、代码分割、图片优化
