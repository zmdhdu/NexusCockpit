### NexusCockpit
- 定义：本项目的主产品名称，指代整个车载语音 Agent 平台。命名含义：Nexus（枢纽/连接点）+ Cockpit（驾驶舱/控制台），体现其作为多座舱智能交互中枢的定位。
- 别名：Nexus 座舱、Nexus 控制台

### 多座舱 CS 架构
- 定义：v2.1 版本的核心架构设计，CS 代表 Client-Server 模式。支持 3 个座舱并行管理，每个座舱拥有独立的数据隔离和状态追踪，通过 Go 网关统一接入。
- 别名：多座舱架构、CS 架构、Multi-Cockpit

### Supervisor + Expert 多智能体
- 定义：基于 LangGraph 的多 Agent 协作模式。1 个 Supervisor（监督者）负责意图识别和任务分派，5 个 Expert（专家）并行执行具体任务（车控、导航、生活、健康、闲聊），最后由 Responder 汇总结果。
- 别名：多 Agent 协作、Supervisor 模式、Expert 并行

### GraphRAG 三路融合
- 定义：检索增强生成的融合策略，同时从三个路径召回相关信息：向量路径（Milvus 语义搜索）、图谱路径（Neo4j 关系推理）、全文路径（BM25 关键词匹配），通过 RRF（Reciprocal Rank Fusion）算法融合排序。
- 别名：GraphRAG、三路召回、RRF 融合

### 副作用隔离
- 定义：安全设计机制，标记为 has_side_effect=True 的操作（如车控指令）永远不会被语义缓存，确保每次真实执行。防止误操作被缓存复用导致的安全风险。
- 别名：副作用保护、has_side_effect

### 双模式部署
- 定义：支持本地 Docker 部署和云端托管两种模式的架构设计。通过 Provider 工厂模式和环境变量切换，同一套代码可在本地 Milvus/Neo4j/Redis 和云端 Zilliz/AuraDB/云 Redis 之间无缝切换。
- 别名：local/cloud 模式、Provider 切换、混合部署

### SubAgent 三层降本监控
- 定义：独立的监控 Agent 体系，通过三层过滤机制降低 LLM 调用成本：Layer 1 规则匹配、Layer 2 异常模式匹配（基于异常类型+指标范围）、Layer 3 LLM 语义理解。通过 Redis Pub/Sub 上报异常给 MainAgent 确认层。
- 别名：SubAgent 监控、三层降本、异常监控

### MainAgent 确认层
- 定义：安全管控机制，在 SubAgent 检测到异常后，由 MainAgent 进行最终确认才放行操作。防止误报导致的误操作，提高系统的可靠性。
- 别名：MainAgent 确认、安全确认层、确认机制

### 车控总线
- 定义：车控指令的统一抽象层，支持三种适配器模式：mock（模拟模式）、HTTP REST（真实车机 API）、MCP stdio（标准输入输出协议）。通过 VehicleAdapter 工厂模式动态选择。
- 别名：Vehicle Bus、车控适配、VehicleAdapter

### RBAC 四级权限
- 定义：基于角色的访问控制体系，定义了四个权限级别：super_admin（超级管理员）、admin（座舱管理员）、cockpit_user（座舱用户）、guest（访客）。Go 网关中根据角色分配不同的 API 访问权限。
- 别名：RBAC 权限、角色权限、四级权限

### 令牌桶限流
- 定义：基于 Redis 实现的滑动窗口限流算法，支持三级优先级：高优先级（核心功能）、中优先级（普通功能）、低优先级（辅助功能）。共享同一个令牌池，令牌不足时低优先级先被拒绝。
- 别名：令牌桶、滑动窗口限流、三级限流

### 语义缓存
- 定义：基于向量相似度的回答缓存机制。将用户输入的 embedding 与缓存中的历史回答 embedding 计算余弦相似度，超过阈值（默认 0.92）则直接复用缓存回答，减少重复 LLM 调用成本。
- 别名：向量缓存、语义复用、Semantic Cache

### 防幻觉四层机制
- 定义：防止 LLM 产生虚假信息的四道防线：预校验（输入合法性检查）、Tool→LLM（工具调用验证）、反思校验（CoT 自我批评）、后校验（输出质量检查）。层层递进确保回复的真实性。
- 别名：防幻觉、幻觉防护、四层校验

### MCP 协议
- 定义：Model Context Protocol，标准化的模型上下文协议。项目中主要用于车控适配器，通过 stdio 模式与外部 MCP 服务通信，实现工具能力的标准化接入。
- 别名：MCP stdio、模型上下文协议、MCP 适配器

### LangGraph StateGraph
- 定义：LangChain 生态中的有状态图编排框架。项目基于 StateGraph 构建 Supervisor 工作流，支持条件路由、状态持久化（checkpoint）、并行执行等特性。
- 别名：StateGraph、LangGraph 图、有状态图
