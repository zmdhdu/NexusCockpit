本模块按用途划分为三个并列的根目录，均作为只读资源被上层服务通过环境变量路径引用：
- `data/`：应用运行期数据，含食物知识库 (`food/*.json`)、知识文档 (`knowledge/*.md`)、用户上传 (`uploads/`)、临时缓存 (`temp/`) 及默认偏好 (`preferences/default_user.json`)；除 `.gitkeep` 外由 `.gitignore` 排除版本控制，路径通过 `FOOD_DATA_DIR`、`KNOWLEDGE_DATA_DIR`、`UPLOAD_DIR`、`TEMP_DIR` 注入。
- `assets/`：静态音频资源，`speaker/enroll_wav/` 与 `speaker/users/` 存放声纹注册样本（随 Git 跟踪），`audio/music/` 提供示例音乐，`audio/prompts/` 用于 TTS 零样本克隆参考。
- `models/`：离线推理模型权重，按能力分子目录：`asr/sensevoice/` (SenseVoice 多语言识别)、`llm/llama.cpp/` (ggml/gguf 运行时 + Qwen3.5-4B GGUF)、`reranker/bge-reranker-v2-m3/` (跨语种重排)、`sv/cam_plus/` (Cam++ 说话人验证)、`tts/cosyvoice/` (CosyVoice 语音合成)；各子目录自带 `README.md` / `configuration.json` / `config.yaml` 描述加载参数，LLM 默认走云端 Ark API，仅在 `USE_LOCAL_LLM=true` 且 `LOCAL_LLM_MODEL_PATH=./models/llm/qwen` 时启用本地 GGUF。
