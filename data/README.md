# Data 目录

本目录存放项目运行所需的数据文件。

## 子目录

| 目录 | 说明 | 环境变量 |
|------|------|----------|
| `food/` | 食物知识库 (JSON/CSV) | `FOOD_DATA_DIR` |
| `knowledge/` | 知识文档 (txt/md/pdf) | `KNOWLEDGE_DATA_DIR` |
| `uploads/` | 用户上传的文件 | `UPLOAD_DIR` |
| `temp/` | 临时文件 (运行时生成) | `TEMP_DIR` |

> 除 `.gitkeep` 外，此目录下的文件不会被 Git 跟踪。
